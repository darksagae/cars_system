# 🚛 SHEET 5: 7-19.9 TONNE TRUCKS - COMPLETE GUIDE

## 🎯 WHAT IS THIS SHEET?

Sheet 5 "7 TONNE TRUCKS" is a **TAX CALCULATOR** for **MEDIUM-HEAVY TRUCKS** weighing 7 to 19.9 tonnes.

**Think of it as:** The "sweet spot" truck category with BETTER tax rates than lighter trucks!

---

## 🔑 KEY ADVANTAGES OF THIS CATEGORY

```
╔════════════════════════════════════════════════════════════════╗
║     WHY 7-19.9 TONNE TRUCKS HAVE BETTER TAX RATES            ║
╠════════════════════════════════════════════════════════════════╣
║                                                                ║
║  🟢 LOWER Import Duty: 10%                                     ║
║     vs 20-25% for lighter trucks (Sheet 4)                    ║
║     SAVES: 10-15% of customs value                            ║
║                                                                ║
║  🟢 LOWER Registration Fees: 1,250,000 UGX                     ║
║     vs 1,500,000 UGX for light vehicles                       ║
║     SAVES: 250,000 UGX per vehicle                            ║
║                                                                ║
║  🟡 Same Environmental: 20%                                    ║
║     (all trucks pay 20%, regardless of tonnage/age)           ║
║                                                                ║
║  ✅ Total Tax Rate: ~58% of CIF                                ║
║     vs ~71-75% for light trucks                               ║
║                                                                ║
╚════════════════════════════════════════════════════════════════╝

BUSINESS INSIGHT:
Heavier trucks = LOWER taxes!
Recommend 7+ tonne if customer has capacity needs!
```

---

## 📋 COMPLETE TAX FORMULAS

```
Row 3:  Exchange Rate        = Linked to 'Trucks & Cabins'!B3
Row 6:  Customs Value        = B5 × B3
Row 8:  Import Duty          = B6 × 10%  🟢 ONLY 10%!
Row 9:  V.A.T                = (B6 + B8) × 18%
Row 10: W.H.T                = B6 × 6%
Row 11: Environmental Levy   = B6 × 20%  (age-independent)
Row 12: Registration Fees    = 1,250,000 🟢 LOWER!
Row 13: Stamp Duty          = 18,000
Row 14: Form Fees           = 35,000
Row 15: Infrastructure Levy  = B6 × 1.5%
Row 16: TOTAL TAXES         = SUM(B8:B15)
```

---

## 💰 WORKED EXAMPLE: 2012 Isuzu FTR (10 Tonnes)

```
Vehicle: 2012 Isuzu FTR (13 years old)
Weight: 10 tonnes
CIF: $20,000
Exchange Rate: 3,800 UGX/USD

Customs Value: 76,000,000 UGX

TAX BREAKDOWN:
─────────────────────────────────────────────────────────
Import Duty (10%):        7,600,000 UGX 🟢
V.A.T (18%):             15,048,000 UGX
W.H.T (6%):               4,560,000 UGX
Environmental (20%):     15,200,000 UGX
Infrastructure (1.5%):    1,140,000 UGX
Registration:             1,250,000 UGX 🟢
Stamp + Form:                53,000 UGX
─────────────────────────────────────────────────────────
TOTAL TAXES:             43,851,000 UGX (57.7% of CIF) ✅

GRAND TOTAL: 121,965,300 UGX ($32,096 USD)
```

---

## 🔍 COMPARISON: ALL TRUCK WEIGHTS

```
Same $20,000 CIF, Different Tonnage:

Light Truck 3T (Sheet 4):     54M taxes → 133M total
Medium-Heavy 10T (Sheet 5):   44M taxes → 122M total 🟢
Super Heavy 25T (Sheet 6):    33M taxes → 111M total 🟢🟢

💡 Heavier = Cheaper! Save 10M+ by going bigger!
```

---

## 🎯 KEY INSIGHTS

1. **10% Import Duty** - Half of light trucks!
2. **1.25M Registration** - 250K cheaper than cars
3. **20% Environmental** - Same as all trucks (no age penalty)
4. **~58% Total Tax** - Most efficient truck category
5. **Minimum Tax: 1,303,000 UGX** - When CIF = 0

**Status:** ✅ Complete
