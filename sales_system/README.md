# 🚗 NSB Motors Ug - Car Sales Management System

A modern, professional car sales management system built with Flutter featuring beautiful glassmorphism UI design, specifically designed for NSB Motors Uganda.

## ✨ Features

### 🎨 **Beautiful Glass UI**
- Modern glassmorphism design
- Smooth animations and transitions
- Professional gradient backgrounds
- Responsive layout

### 📊 **Dashboard**
- Real-time sales statistics
- Quick action buttons
- Recent activity feed
- Performance metrics

### 👥 **Customer Management**
- Customer database
- Contact information
- Payment history
- Balance tracking

### 📄 **Invoice System**
- Professional invoice generation
- PDF export
- Automatic numbering
- Custom templates

### 📧 **Communication**
- Email integration
- WhatsApp messaging
- Automated reminders
- Document sharing

### 📈 **Reports & Analytics**
- Sales reports
- Customer analytics
- Payment tracking
- Export capabilities

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Visual Studio Community (for Windows development)
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd sales_system
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Enable Windows support**
   ```bash
   flutter config --enable-windows-desktop
   ```

4. **Run the application**
   ```bash
   flutter run -d windows
   ```

## 🛠️ Build for Distribution

### Create Windows Installer

1. **Build the application**
   ```bash
   flutter build windows --release
   ```

2. **Create installer using Inno Setup**
   - Download Inno Setup from https://jrsoftware.org/isinfo.php
   - Create installer script
   - Generate .exe installer

### Portable Version
- Copy the `build/windows/runner/Release` folder
- Distribute as portable application

## 🔧 Development

### Project Structure
```
lib/
├── main.dart                 # App entry point
├── providers/               # State management
├── screens/                 # UI screens
├── widgets/                 # Reusable components
├── models/                  # Data models
├── services/               # Business logic
└── utils/                  # Utilities
```

### Adding New Features
1. Create new screen in `screens/`
2. Add navigation in `home_screen.dart`
3. Implement business logic in `services/`
4. Update state management in `providers/`
