# Font Awesome Setup Instructions

## Download Font Awesome Fonts

1. Visit [fontawesome.com/download](https://fontawesome.com/download)
2. Download "Font Awesome 6 Free for Desktop" ZIP file
3. Extract the ZIP file
4. Copy these files to `assets/fonts/`:
   - `fa-brands-400.ttf` (for social media icons: WhatsApp, Instagram, Facebook, Twitter/X, TikTok)
   - `fa-solid-900.ttf` (for location/address icon)

## Files Needed
- `assets/fonts/fa-brands-400.ttf`
- `assets/fonts/fa-solid-900.ttf`

## After Adding Fonts
Run: `flutter pub get`

## Icon Codepoints Used
- Location: 0xf3c5 (fa-map-marker-alt)
- WhatsApp: 0xf232 (fa-whatsapp)
- Instagram: 0xf16d (fa-instagram)
- Twitter/X: 0xf099 (fa-twitter)
- Facebook: 0xf09a (fa-facebook)
- TikTok: 0xe07b (fa-tiktok)
- Email: 0xf0e0 (fa-envelope)
