NSB MOTORS SALES SYSTEM
=======================

Thank you for downloading the NSB Motors Sales System!

EASY SETUP:
-----------
1. Double-click on "install_and_run.bat"
2. Follow the on-screen instructions
3. The application will automatically install required components and start

SYSTEM REQUIREMENTS:
-------------------
- Windows 10 or newer (64-bit only)
- At least 2GB RAM
- 200MB free disk space

FILES IN THIS PACKAGE:
----------------------
- install_and_run.bat - Automated setup (recommended)
- sales_system.exe - Main application
- vc_redist.x64.exe - Microsoft Visual C++ Redistributables
- data/ - Application data folder
- Supporting DLL files

TROUBLESHOOTING:
----------------
If you encounter any issues:

1. Make sure you're using a 64-bit version of Windows
2. Try running "install_and_run.bat" as Administrator
3. Check that your antivirus is not blocking the application
4. Ensure you have sufficient permissions to install software

For technical support, please contact the development team.
