NSB MOTORS SALES SYSTEM - COMPLETE INSTALLATION GUIDE
=====================================================

Thank you for choosing the NSB Motors Sales System!

This is a complete installation package that will install the application on your computer.

INSTALLATION INSTRUCTIONS:
-------------------------
1. Copy the "NSB_Motors_Sales_System_Full.zip" file to your computer
2. Right-click on the ZIP file and select "Extract All..."
3. Navigate to the extracted folder
4. Double-click on "install.bat" to start the installation
5. Follow the on-screen instructions

The installer will:
- Install the application to Program Files\NSB Motors
- Install required Visual C++ Redistributables
- Create desktop and Start Menu shortcuts
- Launch the application automatically

SYSTEM REQUIREMENTS:
-------------------
- Windows 10 or newer (64-bit only)
- At least 2GB RAM
- 200MB free disk space

UNINSTALLATION:
---------------
To uninstall the application:
1. Go to Control Panel > Programs and Features
2. Find "NSB Motors" in the list
3. Click "Uninstall"
