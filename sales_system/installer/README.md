# NSB Motors Ug - Vehicle Sales Management System
## Windows 11 Compatible Installer

### System Requirements
- Windows 10/11 (64-bit)
- 4GB RAM minimum
- 500MB free disk space
- Internet connection for initial setup (optional)

### Installation Instructions
1. Right-click on `NSB_Motors_Setup.exe`
2. Select "Run as administrator" (if prompted)
3. Follow the installation wizard
4. Launch from Desktop shortcut or Start Menu

### Security Notes
- This application is digitally signed for Windows 11 compatibility
- Windows Defender may show a warning - click "More info" then "Run anyway"
- The application is safe and does not contain malware
- All data is stored locally on your computer
