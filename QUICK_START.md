# 🚀 QUICK START - TAX AUTO-LOOKUP

## ⚡ 3-MINUTE SETUP

### Step 1: Import Sample Data (2 minutes)

```
Settings → Tax Database Import
   ↓
Click file upload area
   ↓
Select: sample_tax_database.csv
   ↓
Click "Import Tax Database"
   ↓
✅ Done! 10 records imported
```

### Step 2: Test Auto-Lookup (1 minute)

```
Invoices → Create Invoice
   ↓
Enter: Toyota Wish 2012, 1800cc
   ↓
Click purple "Auto" button
   ↓
See: UGX 13,384,200
   ↓
Click "Use This Tax"
   ↓
✅ Done! Tax auto-filled
```

## 🆘 TROUBLESHOOTING

**"Tax Not Found"?**
- Check spelling (Toyota vs TOYOTA)
- Verify year is in range
- Try different engine format
- Or enter manually

**Import Failed?**
- Check file format (CSV, XLSX, XLS)
- Verify columns match template
- Export template for reference
