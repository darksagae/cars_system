## 🚀 HOW TO USE (Quick Reference)

### First Time Setup (5 minutes):

1. **Open App** → Settings → Tax Database Import
2. **Click** file upload area
3. **Select** `/home/darksagae/Desktop/Enick_Sales/sample_tax_database.csv`
4. **Click** "Import Tax Database"
5. **See** "10 records imported" ✓

### Create Invoice with Auto-Lookup (30 seconds):

1. **Go to** Invoices → Create Invoice
2. **Enter** vehicle: Toyota Wish 2012, 1800cc
3. **Click** purple "Auto" button
4. **See** tax breakdown: UGX 13,384,200
5. **Click** "Use This Tax"
6. **Done!** Tax auto-filled

### Monthly Update (5 minutes):

1. **Get** new MV Database file from URA
2. **Go to** Settings → Tax Database Import
3. **Upload** new file
4. **Set** current month
5. **Import** ✓
